# Operations runbooks

| Runbook | When to use |
|---------|-------------|
| [ETL failure](etl-failure.md) | Reporting snapshot job failed or data is stale. |
| [Power BI refresh failure](pbi-refresh-failure.md) | Dataset refresh errors in the Power BI service. |
| [Credential rotation](credential-rotation.md) | JWT, database, or BI read-only user rotation. |
